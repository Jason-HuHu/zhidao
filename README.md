`知道 `是一个简单的在线问答网站，寻找你想知道的一切，答出你知道的一切

1. 用户注册、登录
2. 问题发布、问题排序、问题分类
3. 图片上传及管理
4. 首页timeline展示、问题广场展示
5. 数据抓取及内容填充
6. 用户关注，粉丝列表
7. 用户评论、用户点赞、用户点踩
8. 邮件通知、站内信通知

# 技术选型
Spring-boot + mybatis + mysql +redis + solr
